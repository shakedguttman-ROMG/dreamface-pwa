# DreamFace — אפליקציית החלפת פנים על-ידי מכשיר

אפליקציית PWA שמריצה **החלפת פנים** (תמונה או סרטון) ישירות על הטלפון שלך.
מבוססת על פרויקט ה-DREAMING (Python/InsightFace) — כאן הומרה לגרסת דפדפן שרצה ב-100% במכשיר.

## מה זה עושה
1. בוחרים **תמונת פנים** (מקור — הזהות להעתקה)
2. בוחרים **תמונה או סרטון** (יעד — הפוזה/הגוף/ההבעה)
3. לוחצים "החלף פנים" → המכשיר מנתח ומחליף
4. אם רוצים — מורידים את התוצאה. **כלום לא נשלח לשרת.**

## איכות
- **נתיב נוירוני (איכות מקסימלית):** מודל `inswapper_128.onnx` (אותו מודל שמשתמש בו ה-DREAMING).
  נטען בפעם הראשון מ-CDN ונשמר ב-cache של הדפדפן → פעמים הבאות עובד **אופליין**.
  מעתיק את המרקם, הקמטים, המיקרו-פרטים וההבעה בצורה טבעית.
- **נתיב גיבוי (קל):** אלגוריתם JS (68 נקודות ציון + Delaunay warp) — עובד גם בלי המודל הנוירוני.

## התקנה על Galaxy S26 Ultra
1. פתח את Chrome וגלוש לכתובת ה-Pages (תינתן בנפרד).
2. לחץ על ⋮ (תפריט) → **הוסף למסך הבית** (Add to Home screen).
3. האפליקציה תופיע על המסך כמו אפליקציה רגילה — פותחים אותה משם.
4. בפעם הראשונה (מקוון): המתן שהמודל הנוירוני יירד (~530MB). זה קורה פעם אחת.

## פרטים טכניים
- `index.html` — ממשק RTL עברית, מותאם מובייל
- `app.js` — מנוע היברידי (נוירוני + גיבוי)
- `manifest.webmanifest` + `sw.js` — PWA, עבודה אופליין
- `vendor/` — face-api.js, delaunator, onnxruntime-web (wasm)
- `models/` — מודלי זיהוי פנים (TinyFaceDetector + 68 landmarks)

## הערות פרטיות
- כל העיבוד רץ במכשיר. תמונות/סרטונים לא עוזבים את הטלפון.
- הסרטון שומר רק את מה שבוחרים להוריד.
- עיבוד סרטון איטי יותר (רץ על CPU של הטלפון) — מומלץ קטעים קצרים.
